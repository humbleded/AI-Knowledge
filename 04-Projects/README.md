## 分类规则

- 先按知识点主题域分层，例如 `Python`、`LLM`、`Agent`、`MCP`、`RAG`。
- 主题域下面再放项目、学习执行区或源码阅读对象。
- 项目目录内可以按任务编号、阶段或功能写笔记。
- 课程资源索引放 `03-Courses`；只有涉及运行、代码、源码阅读、实验结果时才放 `04-Projects`。
- 项目复盘如果是阶段总结，放 `07-Reviews`，并从项目笔记反向链接过去。
