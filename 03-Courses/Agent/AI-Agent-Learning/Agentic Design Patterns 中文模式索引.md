## 为什么有用

这是 Agent 设计模式的中文翻译项目，适合 D7 阶段使用。它能把“我会调用模型”提升到“我知道什么场景该用什么 Agent 结构”。

## 核心模式分组

### 基础编排模式

- Prompt Chaining：提示链
- Routing：路由
- Parallelization：并行化

### Agent 核心模式

- Reflection：反思
- Tool Use：工具使用
- Planning：规划
- Multi-Agent Collaboration：多智能体协作

### 可靠性与工程模式

- Memory Management：记忆管理
- Knowledge Retrieval：知识检索 / RAG
- Model Context Protocol：MCP
- Human-in-the-Loop：人机协同
- Guardrails / Safety Patterns：安全防护
- Evaluation and Monitoring：评估与监控

## 使用方式

每学一个模式，沉淀一张概念卡，固定回答：

- 这个模式解决什么问题？
- 什么时候该用？
- 什么时候不该用？
- 最小代码结构是什么？
- 它和当前项目有什么关系？
