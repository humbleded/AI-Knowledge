## 为什么有用

Hugging Face agents-course 适合在你具备 Python 和基本 LLM API 使用能力后学习。它把 Agent 基础、框架、Agentic RAG、评估和最终项目串在一起。

本地仓库里有 `units/zh-CN` 目录，可优先读中文版内容；没有中文的部分再看英文并整理中文摘要。

## 学习策略

- 不要一开始就学所有框架。
- 先用 Unit 1 建立 Agent 概念。
- 到 G8 阶段重点学 LangGraph。
- 到 R6 阶段重点学 LlamaIndex 和 Agentic RAG。
- 到 E10 阶段再学 trace、eval、benchmark。

## 和中文资料的搭配
