这是中文 Agent 系统教程，适合在 Python 基础之后作为 AI Agent 主线资料。它覆盖 Agent 概念、经典范式、框架实践、记忆、RAG、上下文工程、MCP/A2A、评估和综合项目。

## 阅读策略

1. 先读第一章到第四章，建立 Agent 基础概念。
2. 再读第七章，把“Agent 框架”拆成代码结构。
3. 学到 RAG、MCP、评估阶段时再回读对应章节。

## 注意

这份资料内容很大，不应该从头到尾一次性读完。它更适合作为每个阶段的中文参考书。
