# Agent-Learning-Hub 中文路线索引

## 为什么有用

这是一个 AI Agent 学习路线仓库，适合当作“总导航”。它的价值不是某个具体教程，而是帮你决定下一阶段应该学什么、做什么产物。

## 中文理解

它强调不要只收藏链接，而要按阶段产出东西：

- 理解什么是 Agent。
- 写最小 Agent Loop。
- 学工具调用、RAG、Memory。
- 研究一个现代 Agent Harness。
- 学多 Agent 协作。
- 学 Skills、MCP、A2A、ACP 等能力包装和协议。
- 学浏览器/计算机使用 Agent。
- 做评估、安全和真实项目。

## 对你的作用

这份路线可以作为 `AI-Agent-Learning` tracker 的上层参考：

- 当前 P0 是前置 Python。
- L1、T3、A4 对应最小 Agent Loop。
- R6 对应 RAG 和 Memory。
- H5 对应现代 Agent Harness。
- M9 对应 MCP。
- E10 和 FINAL 对应评估、安全和综合项目。

## 使用方式

每次完成一个阶段后，回到这份路线检查：

- 我是否只看了资料，还是做出了可运行产物？
- 我的 Agent 是否有工具、状态、错误处理、评估？
- 这个阶段有没有可沉淀到知识库的概念卡或项目复盘？
