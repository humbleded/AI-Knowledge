2. 写一个函数 `rename_task(tasks, task_status, old_name, new_name)` 的核心逻辑。要求：

   - 如果 `old_name` 不在 `tasks`，返回 `"not_found"`。
   - 如果找到了，更新 `tasks` 里的任务名。
   - 如果 `old_name` 在 `task_status` 里，把旧状态迁移到 `new_name`。
   - 最后返回 `"renamed"`。

   ```python
def rename_task(tasks, task_status, old_name, new_name):
    if old_name not in tasks:
        return "not_found"

    index = tasks.index(old_name)
    tasks[index] = new_name

    if old_name in task_status:
        task_status[new_name] = task_status.pop(old_name)

    return "renamed"
   ```
