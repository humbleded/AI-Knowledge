# CS50P Weeks 中文学习索引

来源：

- [CS50P Weeks](https://cs50.harvard.edu/python/weeks/)

## 为什么有用

CS50P 适合作为 P0 阶段的视频主线。它比中文文字教程更强调编程思维、调试习惯和作业训练。

## 中文化学习顺序

| CS50P Week                         | 中文理解         | 对应任务              |
| ---------------------------------- | ------------ | ----------------- |
| Week 0 Functions, Variables        | 函数、变量、输入输出入门 | P0-01、P0-02、P0-05 |
| Week 1 Conditionals                | 条件判断         | P0-03             |
| Week 2 Loops                       | 循环           | P0-03、P0-04       |
| Week 3 Exceptions                  | 异常处理         | P0-07             |
| Week 4 Libraries                   | 库、模块、第三方包    | P0-06             |
| Week 5 Unit Tests                  | 单元测试         | P0-07             |
| Week 6 File I/O                    | 文件读写         | P0-08             |
| Week 7 Regular Expressions         | 正则表达式        | P0 后补充            |
| Week 8 Object-Oriented Programming | 面向对象         | 后续 Agent 框架源码阅读前补 |
| Week 9 Et Cetera                   | 综合补充         | P0 后查漏补缺          |

## 使用方式

- 每周内容只做“与当前任务相关”的部分，不一次性刷完整课。
- 视频看完后要写对应脚本，并放到 `AI-Agent-Learning/code/stage0`。
- 通过 PASS 后再把稳定结论沉淀成知识库概念卡。
