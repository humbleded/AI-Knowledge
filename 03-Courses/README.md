## 分类规则

- 先按知识点主题域分层，例如 `Python`、`LLM`、`Agent`、`MCP`、`RAG`。
- 主题域下面再放课程、教程或学习路线目录。
- 同一学习执行区可以拆到不同知识域。
- 不把练习代码复盘放在这里；代码练习和源码阅读放入 `04-Projects`。
- 当某个课程目录超过 8-10 篇笔记时，再按 `stage0`、`unit1`、`chapter1` 等方式继续分层。
