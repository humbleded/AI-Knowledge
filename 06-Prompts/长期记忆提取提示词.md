# 长期记忆提取提示词

请从下面对话或笔记中提取值得长期保存的内容。

只保留满足以下条件的内容：

- 用户明确表达的长期偏好
- 可复用的工作流
- 重要项目路径或约定
- 以后会影响回答方式的决策
- 经过验证的本地环境事实

不要保存：

- 临时情绪
- 一次性闲聊
- 未验证猜测
- 过期可能性很高的信息

输出格式：

```markdown
# 长期记忆候选

## 应保存

- ...

## 不应保存

- ...

## 推荐放入知识库的位置

- `08-Memory/...`
```
