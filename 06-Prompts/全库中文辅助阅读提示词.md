# 全库中文辅助阅读提示词

用于把英文资料整理成适合 `D:\AI-Knowledge` 的中文辅助阅读内容。

## Prompt

请阅读下面的 Markdown 笔记，为它补充中文辅助阅读内容。

要求：

1. 不要整篇硬翻译。
2. 保留英文原文和技术锚点。
3. 不翻译代码块、inline code、命令、路径、URL、API 名、类名、函数名、变量名。
4. 专业词保留英文，并用括号补中文解释，例如 `backpropagation（反向传播）`。
5. 如果笔记较长，新增 `## 中文速读`，用 5-8 条中文 bullet 总结。
6. 如果术语较多，新增 `## 关键术语`。
7. 如果是报错或日志，保留英文原文，再补中文解释。
8. 不编造原文没有的信息。
9. 保持 Obsidian wikilink 不变。

输出格式：

```markdown
## 中文速读

- ...

## 关键术语

- `term`（中文解释）：一句话解释。
```

原始笔记：

```markdown
在这里粘贴 Markdown 内容
```
