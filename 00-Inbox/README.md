# Inbox

这里放临时收集、尚未判断去处的资料。

## 分拣规则

- 原始资料、网页、字幕、论文摘录 -> [[../raw/README|raw]]
- 单个可复用概念 -> `02-Concepts/主题域/`
- 课程路线、教程、视频系列 -> `03-Courses/知识域/课程名/`
- 代码练习、源码阅读、实验记录 -> `04-Projects/知识域/项目名/`
- 周期性总结、问题复盘 -> `07-Reviews/主题/`
- 长期偏好和可复用规则 -> `08-Memory/`

## 分拣时要做

- 补 frontmatter。
- 加 2-5 个相关 wikilink。
- 必要时更新 [[../index|全局索引]] 或对应地图。
- 完成后在 [[../log|log]] 记录一次。
