# Obsidian 插件配置

## 已安装插件

### Dataview

用途：把 Markdown 笔记当成数据库查询。

适合本知识库：

- 列出所有 `status: seed` 的概念笔记。
- 列出所有课程笔记。
- 建立学习进度面板。

### Templater

用途：快速插入模板。

已配置模板目录：

```text
Templates
```

适合本知识库：

- 新建概念笔记。
- 新建课程笔记。
- 新建周复盘。

### Omnisearch

用途：增强全文搜索。

适合本知识库：

- 快速搜索概念、课程、复盘和提示词。
- 在大量 Markdown 文件中定位内容。

### Kanban

用途：用看板管理任务。

适合本知识库：

- 管理 AI 学习任务。
- 管理 PASS 后需要沉淀的内容。
- 管理资料整理队列。

### Smart Connections

用途：用 embedding 找到相关笔记。

适合本知识库：

- 发现概念之间的隐藏关联。
- 根据当前笔记推荐相关笔记。
- 辅助长期复习。

注意：需要在插件设置中配置模型或 API key。

### Copilot

用途：在 Obsidian 中和知识库对话。

适合本知识库：

- 基于 vault 问答。
- 总结当前笔记。
- 辅助整理 Inbox。

注意：需要在插件设置中配置模型或 API key。

## 当前建议

第一阶段优先使用：

1. Omnisearch
2. Templater
3. Dataview
4. Kanban

等知识库内容变多后，再重点使用：

1. Smart Connections
2. Copilot

## 插件启用列表

插件已下载到：

```text
D:\AI-Knowledge\.obsidian\plugins
```

启用列表已写入：

```text
D:\AI-Knowledge\.obsidian\community-plugins.json
```

如果 Obsidian 已经打开，重启 Obsidian 或重新加载窗口后生效。
