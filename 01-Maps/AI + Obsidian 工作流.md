# AI + Obsidian 工作流

## 工作流总览

```text
收集资料 -> 放入 Inbox -> AI 初步整理 -> 拆分成原子笔记 -> 建立链接 -> 每周复盘 -> 沉淀长期记忆
```

## 1. 收集

所有未处理内容先放进 `00-Inbox`，包括：

- 网页摘录
- 视频笔记
- 课程链接
- AI 对话摘要
- 灵感和问题

## 2. AI 整理

让 AI 把原始内容整理成：

- 一句话总结
- 核心概念
- 关键术语
- 可实践任务
- 仍然不懂的问题
- 推荐链接到哪些已有笔记

## 3. 拆分

长笔记不要长期堆在 Inbox。整理后拆到：

- 概念 -> `02-Concepts`
- 课程 -> `03-Courses`
- 项目 -> `04-Projects`
- 论文 -> `05-Papers`
- 提示词 -> `06-Prompts`
- 长期偏好或重要结论 -> `08-Memory`

## 4. 链接

每篇正式笔记至少包含：

- 上级主题
- 相关概念
- 可实践项目
- 仍然不懂的问题

## 5. 复盘

每周在 `07-Reviews` 生成一篇复盘：

- 本周学习了什么
- 哪些概念已掌握
- 哪些概念需要重学
- 下周要做什么项目
- 哪些内容应该加入长期记忆
