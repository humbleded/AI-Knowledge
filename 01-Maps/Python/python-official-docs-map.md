# Python Official Docs Map

这页把 Python 3 官方手册作为长期参考入口。不要把官方手册全文复制进知识库；只在学到具体知识点后抽成概念卡或课程页。

## 官方入口

- [What's new in Python 3.14](https://docs.python.org/3/whatsnew/3.14.html)
- [Tutorial](https://docs.python.org/3/tutorial/index.html)
- [Library reference](https://docs.python.org/3/library/index.html)
- [Language reference](https://docs.python.org/3/reference/index.html)
- [Python setup and usage](https://docs.python.org/3/using/index.html)

## 使用方式

- Tutorial：用来补基础语法和标准风格。
- Library reference：查标准库、内置函数、异常、数据结构。
- Language reference：查语法边界和精确定义。
- HOWTOs：查某一类主题的实践指南，例如 logging、argparse、regex、urllib。
- What's New：版本升级时看行为变化和弃用项。
- Packaging 与 Typing：做工程化 Python 项目时单独查。
