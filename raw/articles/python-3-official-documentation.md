# Python 3 Official Documentation

## 中文速读

- 这是 Python 3 的官方总入口，适合作为语法、标准库、工程化主题和版本差异的校准源。
- 知识库里不要复制整本手册；更合适的做法是先在官方文档里确认，再把稳定结论沉淀到概念卡、课程页或项目记录。
- 学 Python 基础时优先看 `Tutorial`，做工程或排查边界行为时再查 `Library reference`、`Language reference`、`HOWTOs`。

## 关键术语

- `canonical`（权威基准）：出现语法或行为疑问时应回到这里核对。
- `standard library`（标准库）：Python 自带模块与工具集合。
- `HOWTOs`（专题实践指南）：针对某类任务的实用说明。
- `packaging`（打包与分发）：项目依赖、发布和安装相关主题。
