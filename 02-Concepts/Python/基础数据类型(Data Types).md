# Python 基础数据类型

阶段 0 先掌握四种基础类型：

| 类型 | 适合表示 | 示例 |
|---|---|---|
| `str` | 文本 | 姓名、学习目标 |
| `int` | 整数 | 年龄、次数、天数 |
| `float` | 小数 | 每天学习 1.5 小时 |
| `bool` | 真或假 | 是否完成、是否通过 |

## 选择类型的关键

不要只看用户输入长什么样，要看这个值后面要怎么用。

- 要显示文字：用 `str`
- 要做整数计算：用 `int`
- 要保留小数：用 `float`
- 要表达真假状态：用 `bool`
