# Python 变量

变量是给一个值起名字。

例如：

```python
name = "张三"
age = 20
```

`name` 和 `age` 是变量名，右边的内容是它们保存的值。

## 变量名为什么要有意义

有意义的变量名能让自己和别人快速看懂这个值的用途，后续修改代码时也更不容易弄错。

例如 `minutes` 比 `x` 更容易看懂，`is_beginner` 比 `flag` 更能表达布尔值含义。
