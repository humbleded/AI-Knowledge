# Python Interpreter

Python 解释器负责读取并执行 Python 代码。

当运行一个 `.py` 文件时，解释器会从上到下读取代码，把代码转换成计算机可以执行的操作。如果代码有语法错误或运行错误，解释器会把错误信息显示出来。

## 和交互模式的区别

- 运行 `.py` 文件：适合执行完整程序，代码可以保存并反复运行。
- 交互模式：适合临时测试一两句代码，输入一句执行一句。
